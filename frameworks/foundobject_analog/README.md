# Found Object / Analog

## The Exercise

Design / write / author a story, which unfolds itself by examining a found object.

A USB drive:  
Text / pictures / software / links / diary entries  
Corrupted data / metadata

A bag / a wallet / a notebook / pictures / a diary / etc.
